# YPH 电商H5移动端

基于 Vue3 + Vant 构建的移动端电商商城

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vant 4** - 轻量、可靠的移动端组件库
- **Vite** - 下一代前端构建工具
- **Pinia** - Vue 的状态管理库
- **Vue Router** - Vue 官方路由
- **Axios** - HTTP 客户端

## 功能模块

- ✅ 首页（轮播图、分类、商品列表）
- ✅ 商品分类
- ✅ 商品详情
- ✅ 购物车
- ✅ 用户中心
- ✅ 订单管理
- ✅ 登录/注册

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3001

### 构建生产版本

```bash
npm run build
```

## 目录结构

```
src/
├── api/          # API 接口
├── assets/       # 静态资源
├── components/   # 公共组件
├── layout/       # 布局组件
├── router/       # 路由配置
├── stores/       # 状态管理
├── utils/        # 工具函数
├── views/        # 页面组件
├── App.vue       # 根组件
└── main.js       # 入口文件
```
