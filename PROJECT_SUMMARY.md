## 🏗️ 项目架构

### 技术架构

```
┌─────────────────────────────────────────────────────┐
│                   用户界面层                         │
├──────────────────┬──────────────────────────────────┤
│  管理后台         │  H5移动端                        │
│  Vue3+ElementPlus│  Vue3+Vant                       │
└──────────────────┴──────────────────────────────────┘
                    ↓ HTTP/HTTPS
┌─────────────────────────────────────────────────────┐
│                  API网关层                           │
│          Django REST Framework                       │
│          + JWT Authentication                        │
│          + Swagger Documentation                     │
└─────────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────────┐
│                  业务逻辑层                          │
├──────────┬─────────┬─────────┬──────────────────────┤
│ users    │products │  trade  │  marketing  payment  │
│ 用户模块  │ 商品模块 │ 交易模块 │  营销    支付        │
└──────────┴─────────┴─────────┴──────────────────────┘
                    ↓
┌─────────────────────────────────────────────────────┐
│                  数据持久层                          │
├─────────────────────────┬───────────────────────────┤
│     MySQL 8.0           │      Redis 5.0            │
│   (主数据库)             │   (缓存/Session)          │
└─────────────────────────┴───────────────────────────┘
```

## 📈 数据库设计

### 核心数据表（27张）

**用户相关 (6张)**
- user - 用户表
- user_level - 用户等级
- delivery_address - 收货地址
- user_message - 用户消息
- user_points_history - 积分历史
- user_login_history - 登录历史

**商品相关 (8张)**
- product_category - 商品分类
- product_brand - 品牌
- product - 商品
- product_item - 商品SKU
- product_spec - 规格
- product_spec_value - 规格值
- product_comment - 评论
- product_tag - 标签

**交易相关 (6张)**
- cart - 购物车
- order - 订单
- order_item - 订单商品
- order_logistics - 物流
- order_return - 退款
- order_return_reason - 退款原因

**营销相关 (3张)**
- activity_base - 活动
- coupon - 优惠券
- user_coupon - 用户优惠券

**其他 (4张)**
- payment_order - 支付订单
- favorites_item - 收藏
- product_browse - 浏览历史
- search_history - 搜索历史
- banner - 轮播图
- system_config - 系统配置
